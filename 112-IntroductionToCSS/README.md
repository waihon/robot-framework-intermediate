# Intro to CSS

## Purpose:
Gentle Introduction to CSS

## Learning Objectives:

- Trainees will be able to understand the syntax of a CSS rule
- Trainees will be able to utilize CSS with Selenium
- Trainees will be able to create CSS selectors:
`$selector_id {
  property : value;
}`
